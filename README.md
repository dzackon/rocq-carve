# Rocq CARVe-ing

CARVe is a Rocq library for mechanizing substructural logics and languages. 

## Requirements
- [Rocq Prover](https://rocq-prover.org/) (tested ≥ 8.20 and ≤ 9.0)
- [`hammer` tool for Rocq](https://rocq-prover.org/p/coq-hammer/)

## External links

- D. Zackon, C. Sano, A. Momigliano & B. Pientka. 2025. *[Split decisions: Explicit contexts for substructural languages](https://doi.org/10.1145/3703595.3705888)*. Proc. CPP '25. 257–271.
